# ÖRKEN Design System

## Product
ÖRKEN is Kazakhstan's unified business support portal. It must feel authoritative, clear, humane, and administratively credible. Core journeys: discover support, apply through a guided form, track an application, browse reports/projects/tools, and configure services in a no-code admin studio.

## Visual direction
Adapt the cream-and-cobalt modernist system into a government-grade digital product: structured 12-column grids and crisp dividers, balanced by approachable typography, rounded interactive surfaces, generous whitespace, and calm status colors. Avoid gradients, glassmorphism, stock photography, and ornamental decoration.

- Radius: 16px cards, 12px controls, 999px pills. Admin canvas nodes may use 10px.
- Shadows: only subtle elevation, 0 10px 30px rgba(21,33,27,.08).
- Spacing: 4px base; major section gaps 72–96px; page max-width 1240px.

## Components and interaction
- Sticky white header, 72px, slim trust banner above it.
- Primary buttons are forest green, 44–52px high, sentence case. Accent lime is reserved for highlights and AI.
- Cards use 1px borders and clear hierarchy; hover lifts by 2px and strengthens border.
- Inputs have visible labels, 48px height, strong focus rings, and plain-language helper/error copy.
- Application steps show progress and save state. Conditional questions appear with a short explanation.
- Admin studio is a three-pane workspace: component palette, visual form canvas, properties panel.
- Motion: 160–240ms ease-out; respect reduced motion.
- Responsive: mobile navigation collapses; cards stack; admin studio becomes tabbed; no horizontal page overflow.

## Content
Russian is the primary demo language. Use concise, supportive language. Show realistic Kazakhstan business context, KZT amounts, BIN identifiers, Baiterek subsidiaries, and explicit mock-integration labels where relevant.
